# Student Grade Management System

## Project Purpose
This Java console-based application helps teachers manage student academic records digitally. The system allows users to add students, display all records, search students by ID, and calculate average marks.

## Features
- Add Student
- Display All Students
- Search Student by ID
- Calculate Average Marks

## How to Run

```bash
Create Student class
```
### Compile

```bash
javac Student.java StudentGradeSystem.java
```

### Run

```bash
java StudentGradeSystem
```
